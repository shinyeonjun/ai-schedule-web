# - AI 통화 분석 시스템 프로젝트 설계서

## 프로젝트 개요

**프로젝트명:** - AI 통화 분석 시스템

**필요성 및 목적:**
- 통화 내용에서 일정 정보를 수동으로 추출하고 정리하는 번거로움 해결
- AI를 활용한 자동 일정 추출로 업무 효율성 향상
- 통화 내용을 체계적으로 관리하고 캘린더 연동을 통한 일정 자동화
- 팀원 간 일정 공유 및 협업 효율성 증대

**문제 정의:**
- 통화 후 일정 정보를 수동으로 정리하는 시간 소모
- 여러 참석자가 있는 회의에서 각자 일정을 따로 관리하는 비효율성
- 통화 내용에서 중요한 일정 정보를 놓치거나 잘못 기록하는 문제
- 일정 정보를 캘린더에 수동으로 입력하는 번거로움
- 팀원들에게 일정을 개별적으로 공유하는 복잡성

## 개발 내용 및 주요 기능

### 1. 통화 분석 기능
- 텍스트 파일 업로드 또는 직접 입력을 통한 통화 내용 분석
- OpenAI GPT-4o-mini를 활용한 AI 기반 일정 자동 추출
- 단체일정(group)과 개인일정(personal) 자동 분류
- 참석자, 시간, 장소, 설명 정보 자동 추출

### 2. 일정 관리 시스템
- 분석된 일정의 수정 및 편집 기능
- 분석 세션별 일정 그룹 관리
- 일정 삭제 및 세션 삭제 기능
- Google Calendar 연동을 통한 자동 일정 추가

### 3. 사용자 인증 및 관리
- Google OAuth 2.0 기반 로그인 시스템
- JWT 토큰 기반 인증
- 사용자별 데이터 격리 및 보안

### 4. 이메일 연동
- Gmail API를 통한 일정 공유
- ICS 파일 자동 생성 및 첨부
- 다중 수신자 이메일 발송

### 5. 데이터베이스 관리
- Supabase PostgreSQL을 활용한 데이터 저장
- 사용자별 분석 세션 및 일정 데이터 관리
- Google 토큰 안전 저장
