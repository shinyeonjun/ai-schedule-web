# MUFI Frontend CSS 디렉토리 구조

## 📁 전체 구조
```
frontend/css/
├── DOC/                          # 📋 문서 폴더
│   └── 디렉토리.md              # 현재 파일 - CSS 구조 문서
├── login/                        # 🔐 로그인 페이지 스타일
│   └── login.css                 # 📄 로그인 페이지 전용 스타일 (브랜딩, 폼, 반응형)
└── dashboard/                    # 🎯 대시보드 스타일
    ├── dashboard.css             # 🏗️ 메인 대시보드 스타일 (레이아웃, 공통 컴포넌트)
    ├── dashboard_before.css      # 📜 원본 대시보드 스타일 (참고용)
    └── sections/                 # 📑 섹션별 스타일
        ├── analysis.css          # 📞 통화 분석 섹션 스타일 (업로드, 결과 표시)
        ├── schedules.css         # 📅 일정 관리 섹션 스타일 (일정 목록, 카드)
        └── members.css           # 👥 인원 관리 섹션 스타일 (사용자 그리드, 카드)
```

## 📋 파일별 상세 설명

### 🔐 로그인 스타일
- **login.css**: 로그인 페이지 전용 스타일
  - 브랜딩 섹션 (로고, 기능 소개, 통계)
  - 로그인 폼 (Google OAuth 버튼, 반응형 레이아웃)
  - 색상 테마 (Primary Blue #2563eb, Success Green #059669)

### 🎯 대시보드 스타일

#### 🏗️ 메인 대시보드 (dashboard.css)
- **레이아웃**: 전체 대시보드 구조, 사이드바, 메인 콘텐츠 영역
- **공통 컴포넌트**: 카드, 버튼, 폼, 모달, 토스트 알림
- **네비게이션**: 사이드바 메뉴, 헤더, 사용자 프로필
- **반응형**: 모바일, 태블릿, 데스크톱 대응

#### 📑 섹션별 스타일

**📞 analysis.css (통화 분석)**
- 파일 업로드 영역 스타일
- 분석 결과 표시 스타일
- 로딩 오버레이
- 탭 인터페이스

**📅 schedules.css (일정 관리)**
- 일정 목록 그리드 레이아웃
- 일정 카드 디자인
- 공유 배지, 상태 표시
- 빈 상태 (empty state) 스타일

**👥 members.css (인원 관리)**
- 사용자 그리드 레이아웃
- 사용자 카드 디자인
- 연락처 목록 스타일
- 로딩 플레이스홀더

## 🎨 디자인 시스템

### 🎯 색상 팔레트
- **Primary Blue**: #2563eb (주요 액션, 링크)
- **Success Green**: #059669 (성공 상태, 완료)
- **Warning Orange**: #d97706 (경고, 주의)
- **Background**: #ffffff (메인 배경)
- **Text**: #1f2937 (주요 텍스트)

### 📐 레이아웃
- **Sidebar**: 280px 고정 너비
- **Main Content**: 반응형 (최소 768px)
- **Card**: 16px 패딩, 8px 라운드
- **Spacing**: 8px, 16px, 24px, 32px 시스템

### 🔤 타이포그래피
- **Font Family**: Inter (Google Fonts)
- **Weights**: 400, 500, 600, 700
- **Sizes**: 14px, 16px, 18px, 24px, 32px

## 📱 반응형 브레이크포인트
- **Mobile**: < 768px
- **Tablet**: 768px - 1024px
- **Desktop**: > 1024px

## 🔄 CSS 로딩 순서
1. **dashboard.css**: 메인 레이아웃 및 공통 스타일
2. **sections/analysis.css**: 통화 분석 섹션 스타일
3. **sections/schedules.css**: 일정 관리 섹션 스타일
4. **sections/members.css**: 인원 관리 섹션 스타일

## 🎯 특징
- **모듈화**: 기능별로 독립적인 CSS 파일
- **일관성**: 통일된 디자인 시스템 적용
- **성능**: 필요한 스타일만 로드
- **유지보수**: 섹션별 독립적 수정 가능
- **확장성**: 새로운 섹션 추가 용이
