# MUFI Frontend JavaScript 디렉토리 구조

## 📁 전체 구조
```
frontend/js/
├── DOC/                          # 📋 문서 폴더
│   └── 디렉토리.md              # 현재 파일 - JavaScript 구조 문서
├── login/                        # 🔐 로그인 관련 스크립트
│   └── login.js                  # 📄 로그인 페이지 기능 (Google OAuth, 폼 처리)
├── google_token/                 # 🔑 Google 인증 관련
│   └── (Google OAuth 관련 파일들)
└── dashboard/                    # 🎯 대시보드 관련 스크립트
    └── section/                  # 📑 섹션별 스크립트
        ├── analysis/             # 📞 통화 분석 섹션
        │   └── (파일 업로드, 분석 처리, 결과 표시)
        ├── schedules/            # 📅 일정 관리 섹션
        │   └── (일정 목록, ICS 다운로드, 공유 기능)
        └── members/              # 👥 인원 관리 섹션
            └── (사용자 목록, 연락처 관리)
```

## 📋 파일별 상세 설명

### 🔐 로그인 스크립트
- **login.js**: 로그인 페이지 전용 JavaScript
  - Google OAuth 인증 처리
  - 로그인 폼 유효성 검사
  - 로그인 상태 관리
  - 에러 처리 및 알림 표시

### 🎯 대시보드 스크립트

#### 📞 analysis/ (통화 분석)
- **파일 업로드**: 드래그 앤 드롭, 파일 선택
- **분석 처리**: API 호출, 진행률 표시
- **결과 표시**: 분석 결과 렌더링, 일정 추출
- **에러 처리**: 업로드 실패, 분석 오류 처리

#### 📅 schedules/ (일정 관리)
- **일정 목록**: 데이터베이스에서 일정 조회
- **ICS 다운로드**: 캘린더 파일 생성 및 다운로드
- **공유 기능**: 일정 공유, 이메일 발송
- **필터링**: 날짜, 상태별 일정 필터

#### 👥 members/ (인원 관리)
- **사용자 목록**: 사용자 정보 조회 및 표시
- **연락처 관리**: 연락처 추가, 수정, 삭제
- **검색 기능**: 사용자 검색 및 필터링
- **권한 관리**: 사용자 권한 설정

## 🔧 주요 기능

### 🔐 인증 시스템
- Google OAuth 2.0 인증
- 토큰 관리 및 갱신
- 세션 관리
- 로그아웃 처리

### 📊 데이터 처리
- API 통신 (FastAPI 백엔드)
- JSON 데이터 파싱
- 실시간 데이터 업데이트
- 에러 핸들링

### 🎨 UI 인터랙션
- 모달 다이얼로그
- 토스트 알림
- 로딩 스피너
- 반응형 네비게이션

### 📁 파일 처리
- 파일 업로드 (드래그 앤 드롭)
- 파일 형식 검증
- 진행률 표시
- 다운로드 처리

## 🏗️ 아키텍처

### 📦 모듈 구조
- **기능별 분리**: 각 섹션별로 독립적인 JavaScript 파일
- **공통 모듈**: 인증, API 통신, UI 컴포넌트
- **유틸리티**: 헬퍼 함수, 상수, 설정

### 🔄 데이터 플로우
1. **사용자 액션**: 버튼 클릭, 파일 업로드 등
2. **이벤트 핸들러**: 사용자 액션 처리
3. **API 호출**: 백엔드 서버와 통신
4. **데이터 처리**: 응답 데이터 파싱 및 검증
5. **UI 업데이트**: DOM 조작으로 화면 갱신

### 🛡️ 에러 처리
- **네트워크 에러**: API 통신 실패 처리
- **유효성 검사**: 입력 데이터 검증
- **사용자 피드백**: 에러 메시지 표시
- **복구 메커니즘**: 자동 재시도, 폴백 처리

## 📱 브라우저 호환성
- **모던 브라우저**: Chrome, Firefox, Safari, Edge
- **ES6+ 지원**: Arrow functions, async/await, modules
- **폴리필**: 필요한 경우 구형 브라우저 지원

## 🚀 성능 최적화
- **지연 로딩**: 필요한 스크립트만 로드
- **캐싱**: API 응답 캐싱
- **디바운싱**: 검색 입력 최적화
- **메모리 관리**: 이벤트 리스너 정리

## 🎯 특징
- **모듈화**: 기능별로 독립적인 JavaScript 파일
- **재사용성**: 공통 컴포넌트 분리
- **확장성**: 새로운 기능 추가 용이
- **유지보수**: 명확한 파일 구조와 주석
- **테스트**: 단위 테스트 가능한 구조
