## 🎯 핵심 파일 설명

### 📄 HTML
- **login.html**: 로그인 페이지
- **dashboard.html**: 빌드된 최종 대시보드
- **dashboard/main.html**: 메인 레이아웃 템플릿
- **dashboard/sections/**: 각 섹션별 템플릿

### 📄 CSS
- **dashboard.css**: 대시보드 전체 스타일 (기존 유지)
- **login.css**: 로그인 페이지 스타일

### 📄 JavaScript (폴더별 분리)
- **login/login.js**: 로그인 로직
- **google-auth/google-auth.js**: 구글 OAuth
- **analysis/analysis.js**: 통화 분석 기능
- **schedules/schedules.js**: 일정 관리 기능
- **members/members.js**: 인원 관리 기능
- **dashboard.js**: 대시보드 메인 로직
- **api.js**: API 통신
- **utils.js**: 공통 유틸리티

## 🔧 개발 순서

### 1단계: 기본 구조
1. **HTML**: dashboard/main.html (메인 레이아웃)
2. **CSS**: dashboard.css (기존 유지)
3. **JavaScript**: dashboard.js (메인 로직)

### 2단계: 기능별 구현
1. **로그인**: login.html, login/login.js
2. **구글 인증**: google-auth/google-auth.js
3. **통화 분석**: sections/analysis.html, analysis/analysis.js
4. **일정 관리**: sections/schedules.html, schedules/schedules.js
5. **인원 관리**: sections/members.html, members/members.js

### 3단계: 빌드 및 최적화
1. **빌드 스크립트**: build-dashboard.js
2. **API 통신**: api.js
3. **유틸리티**: utils.js

## 📋 장점
